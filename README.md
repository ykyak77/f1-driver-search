# Sistema de Consulta de Pilotos de Fórmula 1

Este repositório contém um sistema simples de consulta de pilotos de Fórmula 1, onde o usuário pode pesquisar pilotos usando o primeiro nome ou o número do piloto. A aplicação é composta por um front-end em HTML, CSS e JavaScript, que se comunica com uma API backend construída com Express e Axios. 

## 🏎️ Estrutura do Projeto

A estrutura do projeto é a seguinte:

- **/img**: Pasta contendo as imagens usadas no frontend (ícone e ícone de lupa para o botão de pesquisa).
- **style.css**: Arquivo de estilo para a interface do usuário.
- **script.js**: Arquivo JavaScript que lida com a lógica do frontend.
- **index.html**: Arquivo HTML principal, onde a interface do usuário é exibida.
- **app.js**: Arquivo backend em Express que realiza a requisição à API de pilotos de Fórmula 1 e retorna os dados.
- **package.json**: Arquivo de configuração do Node.js, onde as dependências são listadas.


#  🚀 Como Rodar o Projeto

## Backend (API)

1. Clone este repositório em seu computador:

    ```bash
    git clone https://github.com/seuusuario/seurepositorio.git
    cd seurepositorio
    ```

2. Instale as dependências do backend:

    ```bash
    npm install
    ```

3. Inicie o servidor backend:

    ```bash
    npm start
    ```

    O servidor será iniciado na porta 3000, a menos que você defina outra porta no ambiente.

## Frontend

O frontend é servido diretamente pela aplicação Express, através da pasta `/Projeto`. Certifique-se de que o servidor backend esteja rodando, pois ele irá servir o HTML, CSS e JavaScript. 

1. Acesse o sistema de consulta pela URL:

    ```
    http://localhost:3000
    ```

2. Utilize os campos de pesquisa para procurar pilotos pelo primeiro nome ou número do piloto.

# 🔧 Funcionalidades

- **Busca por Primeiro Nome**: Você pode pesquisar pilotos inserindo o primeiro nome na caixa de texto. Ex: Max
- **Busca por Número do Piloto**: Também é possível realizar a pesquisa usando o número do piloto. Ex: 5
- **Exibição de Dados do Piloto**: Após a pesquisa, serão exibidos dados como nome completo, equipe, número do piloto e a foto, se disponível.
- **Interação com a API OpenF1**: O backend faz uma requisição à API OpenF1 para recuperar os dados dos pilotos.
